How this works

On Refresh, the server looks up the HTTP IP it sees with ipapi.is and cross-checks the country against ipwho.is and geojs.io, adds a reverse-DNS and blocklist check, and caches the result. Your browser separately probes its IPv4 and IPv6 exits, runs a DNS-leak test, inspects WebRTC candidates via a public STUN server, and estimates a fingerprint. Fingerprint data is computed entirely in your browser and never sent back.

What we don't do

This app has no database and the application code does not write request logs. The hosting platform may still keep operational logs. It sets no cookies and runs no ad trackers. Page visit counts use Umami's cookieless analytics. Every check runs on demand.
